Introducción

Este proyecto consiste en el diseño y automatización de una incubadora para huevos de ganso. Se busca controlar variables como temperatura, humedad y volteo de huevos de forma automática.

Componentes Utilizados

  • Microcontrolador (Arduino / ESP32)
  • Sensor de temperatura y humedad (DHT22)
  • Relevadores o SSR
  • Resistencias calefactoras
  • Motor para volteo de huevos
  • Pantalla LCD / Display
  • Fuente de alimentación

Funcionamiento del Sistema

El sistema monitorea constantemente la temperatura y humedad dentro de la incubadora. Si la temperatura baja, activa las resistencias. El motor realiza el volteo de huevos cada cierto tiempo. Todos los datos se visualizan en una pantalla.

Resultados Esperados / Obtenidos

El sistema mantiene condiciones óptimas para la incubación, mejorando la tasa de eclosión. Aquí se pueden agregar gráficos, fotos o estadísticas del proceso.

Conclusiones y Mejoras Futuras

Este proyecto demuestra cómo la automatización puede mejorar procesos agrícolas. Futuras mejoras incluyen conectividad WiFi, app móvil y panel solar para autosustentabilidad.
